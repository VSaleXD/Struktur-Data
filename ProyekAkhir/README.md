# Proyek Akhir Struktur Data: Sistem Inventori Gudang

## Struktur Folder
- spesifikasi.md : Spesifikasi sistem & skenario
- inventori_array.cpp : Implementasi struktur data (unordered map)
- dataset_dummy.txt : Dataset dummy produk
- test_insert_search.cpp : Kode pengujian insert & search
- hasil_uji.md : Dokumentasi hasil pengujian
- topik.md : Topik & pembagian tugas
- README.md : Petunjuk umum proyek

## Build & Run
- Kompilasi: g++ namafile.cpp -o namafile.exe
- Jalankan: ./namafile.exe

## Konvensi Kode
- 1 file 1 topik, tanpa header
- Gunakan struct & array, bukan STL
- Input/output standar (scanf/printf/cin/cout)
- Indentasi 4 spasi, tanpa tab

## Pembagian Tugas
- Spesifikasi sistem & skenario: Wardah
- Implementasi struktur data: Ale
- Dataset dummy: Azka
- Pengujian insert & search: Anaqi

## Catatan
- Dataset dummy bisa berupa file teks atau hardcode di kode
- Pengujian insert & search dibuat terpisah
- Untuk final, tambahkan file baru untuk struktur data lain
